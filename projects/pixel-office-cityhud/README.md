# Pixel Office CityHUD

这是本仓库里「像素风城市 + 多 Agent 办公室」原型的独立项目入口，用于集中查看产品/设计/实现文档与代码位置。

## 入口
- 原型代码：`projects/pixel-office-cityhud/pixel-prototype/`（指向 `frontend/pixel-prototype/` 的目录联接）
- 文档索引：`projects/pixel-office-cityhud/docs/INDEX.md`

## 快速体验
- 开发启动：在 `projects/pixel-office-cityhud/pixel-prototype/` 下运行 `npm run dev`
- 地图页：`/map/prod`（右侧办公室派工，左侧城市出现扫描/施工投影）
